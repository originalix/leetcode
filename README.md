## 本地运行

克隆此项目

```bash
  git clone https://github.com/originalix/leetcode.git
```

前往项目目录

```bash
  cd leetcode
```

安装依赖

```bash
  npm install
  or
  yarn
```

启动

```bash
  npm run dev
  or
  yarn run dev
```

  
## 单元测试

运行单元测试，执行以下命令

```bash
  npm run test
  or
  yarn run test
```
